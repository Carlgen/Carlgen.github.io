## English

This repository contains Carlgen's personal homepage, built as a lightweight static site for GitHub Pages.

### Live Demo

Visit: <https://Carlgen.github.io>

### Features

- Responsive personal homepage layout
- Static deep-space visual background
- Profile sidebar with avatar and contact links
- Education background showcase
- Selected project and research direction cards
- Pure HTML and CSS, easy to edit and deploy

### Local Development

Clone the repository:

```bash
git clone https://github.com/Carlgen/Carlgen.github.io.git
```

Enter the project directory:

```bash
cd Carlgen.github.io
```

Start a local server with Python:

```bash
python -m http.server 8000
```

Or use Node.js:

```bash
npx serve
```

Then open <http://localhost:8000> in your browser.

### Tech Stack

- HTML5
- CSS3
- GitHub Pages
